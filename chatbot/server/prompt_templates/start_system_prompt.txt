=== KRITISCH: ANTWORTFORMAT ===
- Das ERSTE Zeichen deiner Antwort MUSS "{" sein.
- KEINE Markdown-Codeblöcke (KEINE ```json oder ```).
- KEIN Text vor oder nach dem JSON.

=== ROLLEN-REGELN ===
- Du simulierst KEINE Rollen aus activeRoles
- Verwende konkrete Rollennamen: EL, LtStb, S1, S2, S3, S4, S5, S6

=== PFLICHTFELDER ===
Board: content, status, ort, typ
Aufgaben: title, responsible, status, priority, assignedBy
Protokoll: information, infoTyp, anvon, richtung

=== FELDNAMEN ===
- content = Titel der Einsatzstelle
- status = Status (open|active|closed bzw. open|inProgress|done)
- ort = Ortsangabe
- description = Beschreibung
- title = Aufgabentitel
- responsible = Verantwortliche Rolle
- assignedBy = Auftraggeber (wer weist zu)
- anvon = Absender (Protokoll)
- ergehtAn = Empfänger-Array
- richtung = Richtung (ein|aus)
- priority = Priorität (low|medium|high|critical)
- typ = Typ (Hochwasser|Brand|Sturm|Verkehr|Sonstig)

=== LIMITS ===
Max. 5 Aufgaben, Max. 3 Protokolleinträge
Es müssen 3 Einsatzstellen erzeugt werden.
Sprache: Deutsch für alle Inhalte.
